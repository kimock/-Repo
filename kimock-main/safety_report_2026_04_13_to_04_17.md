## 4. 🌳 종합 위험 구조 진단 (Root Cause Tree)

> ⚠️ 이번 주 전체 Safety Phrase가 소표본(5건 미만) 구간에 해당. 빈도 5건 이상 항목 없음 **(데이터 집계, 섹션 8)**.  
> 최고 빈도(4건) 항목을 대상으로 분석 수행 — 과대해석 금지, 패턴 참고 수준으로 활용.

```
[L1] 고소 개구부 추락·낙하 복합 위험
     — unsafe opening 4건(▲2), safety harness 4건(─) (데이터 집계)
│
├── [L2] 방호 구조물 미설치·규격 미달
│   — opening height 4건, unsafe opening height 4건 ─ (고착) (데이터 집계)
│   └── [L3] 개구부 방호 설치 기준 체크리스트 부재
│          / 개구부 현황 도면 관리·일일 점검 책임자 미지정 (추정)
│
├── [L2] 안전대(Safety Harness) 미착용·앵커포인트 불량
│   — safety harness 4건, anchoring point 2건, using same 2건 (데이터 집계)
│   └── [L3] PTW 연계 착용 확인 시스템 미운영
│          / 2인 공유 앵커포인트 금지 교육 미실시 (추정)
│
└── [L2] 안전 접근로 미확보 반복
    — unsafe access 2건 (신규 등장, 전주 지적 후 미개선) (데이터 집계)
    └── [L3] 작업 전 접근로 확인 프로세스 부재
           / 지적 후 이행 확인 절차 누락 (추정)


[L1] 감전·합선 위험 클러스터
     — 감전/합선 8건(17.0%), binding wire 4건(▲1), electric cable 3건(─) (데이터 집계)
│
├── [L2] 전기 케이블 결속선(Binding Wire) 고정
│   — binding wire 4건, cable tied~tied binding wire 연관군 2건씩 (데이터 집계)
│   └── [L3] 케이블 고정 기준(절연 클램프 사용) 미교육
│          / 결속선 대체재 공급 체계 부재 (추정)
│
├── [L2] 접지(Body Earthing) 미연결
│   — body earthing 2건 ─ (고착), 용접기·그라인더 대상 (데이터 집계)
```
