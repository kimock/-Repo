## 5. 💡 우선순위 개선 권고

### 단기 (즉시~1주)

1. **미완료 14건 즉각 조치 (섹션 12 데이터 집계)**
   - Hankuk 9건: `unsafe manner`(4건) — 행위 특정 후 작업자·감독자 공동 시정 확인
   - DSHI 5건: `lifeline rope`(2건) — Module 2F 생명줄 설치 현황 전수 확인

2. **Scissor Lift 전수 점검 (DSHI Module 2F 우선)**
   - 운용 전 체크리스트 완료 서명 의무화; 미이행 시 장비 운용 중단

3. **Safety Harness 착용 현장 불시 점검 (Hankuk)**
   - 개구부·가장자리 근접 작업 구역(`near edge`) 안전대 앵커포인트 추가 설치

4. **Green Net 위 자재 적치 즉시 제거 (Hankuk)**
   - 낙하물 방지망 기능 복원 후 추가 적치 금지 표지 부착

5. **Daemyoung 전단 스터드 노출 구역 격리** — 데크 시트 공사 완료 전 출입 통제

### 중기 (1개월 이내)

1. **PTW 체크리스트 개편**
   - 고소 작업(>2 m): 하네스 앵커포인트 지정 및 착용 확인란 추가
   - Scissor Lift: 일일 점검 기록 미완료 시 PTW 발급 차단

2. **Tool Box Talk (TBT) 실효성 강화**
   - 단순 구두 교육 → 반복 위반 사례 영상 + 즉각 피드백 방식으로 전환
   - `unsafe manner` 위반자 별도 재교육 이수 후 현장 복귀 원칙 수립

3. **Module 2F JHA 갱신**
   - 공정 이동 시 JHA 의무 갱신 프로세스 확립; 주간 안전회의에서 결과 공유

4. **Scaffold 태그 시스템 강화**
   - 미완성 scaffold `without tag` → Red Tag 부착 및 접근 차단 물리적 조치

5. **DSHI 접지(Earthing) 관리 강화**
   - 용접 장비 접지선 일일 점검 서명 의무화; 감전 위험 구역 경보 표지 확충

### 장기 (분기 이상)

1. **행동 기반 안전 관리(BBS: Behavior-Based Safety) 시스템 도입**
   - `unsafe manner` 반복 위반 근본 해결을 위한 관찰·피드백 루프 프로그램 구축

2. **디지털 위험 모니터링 강화**
   - Scissor Lift·고소 장비에 IoT 센서 부착 → 과적재·앵커링 해제 실시간 알림

3. **Vendor 안전 역량 평가 체계 신설**
   - 월별 위반 건수·반복 패턴을 Vendor 계약 갱신 평가 지표에 포함
   - Hankuk 추락 위험 집중(2주 연속) 개선 KPI 설정 및 주간 보고 의무화

4. **공정 이동 안전 프로세스 표준화**
   - Module 간 공정 이동 전 JHA 갱신 → 안전관리자 서명 → 투입 허가 순서 제도화
